# Sleek Promises

Promises api for sleekxmpp based on the [Promises/A+ Spec](https://promisesaplus.com/#notes).

## Requirements

* SleekXMPP 1.3.1 
* mock 1.3.0
* unittest

## Documentation and Testing 

To run the test suite for sleek promises:

```bash
python -m unittests discover
```
